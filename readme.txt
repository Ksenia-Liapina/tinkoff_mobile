Слева на панели Навигация расположен список разделов сценариев тестирования.
По нему легко передвигаться, ориентируясь на названия сценариев.
Список с вложенными элементами, которые можно закрывать и открывать в зависимости от того, что ищется

Список расширяемый
Новы раздел добавляется с помощью стиля "Заголовок 1".
Подраздел добавляется с помощью стиля "Заголовок 2"
Список сценариев в блок объекта тестирования добавляется с помощью стиля "Заголовок 3"
Каждый сценарий блока отображается нумерованным списком с помощью стиля "Заголовок 4"
